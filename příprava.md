## 1. Příprava projektu

1. Vytvoř si složku projektu, např. `vehicles-orm`:
    ```bash
    mkdir vehicles-orm
    cd vehicles-orm
    ```

2. Otevři terminál a spusť tyto příkazy pro inicializaci Composeru a instalaci Eloquent ORM:
    ```bash
    composer init -n
    composer require illuminate/database
    ```

Tím si připravíš základní prostředí pro práci s Eloquentem v PHP (bez Laravelu).
